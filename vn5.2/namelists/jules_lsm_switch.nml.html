6.2. jules_lsm_switch.nml¶

This file contains a single namelist called JULES_LSM_SWITCH that indicates which land surface model is to be used - JULES or CABLE.

Note

The CABLE model has not yet been implemented within the JULES repository.

6.2.1. JULES_LSM_SWITCH namelist members¶

JULES_LSM_SWITCH::lsm_id¶
Type:integer
Default:0

Switch for land surface model.

1
The JULES land surface model will be used.
2
The CABLE land surface model will be used.
